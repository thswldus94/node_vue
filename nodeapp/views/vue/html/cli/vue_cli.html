2. 프로젝트 만들기

먼저 node 가 설치되어 있어야 한다! node 기반으로 하기 때문... node 가 깔려 있다는 가정 하에서 시작할게요.

왜냐면 내 서버는 node로 되어있으니까 node는 이미 준비되어있어요 ㅎㅎ

- vue init webpack (프로젝트명)

- 질문폭탄

? Project name (프로젝트명)
? Project description A Vue.js project
? Author thswldus1994@naver.com
? Vue build standalone
? Install vue-router? Yes
? Use ESLint to lint your code? No  // 안쓸거니까
? Setup unit tests with Karma + Mocha? No  // 안쓸거니까
? Setup e2e tests with Nightwatch? No  // 안쓸거니까

- 그 후에 localhost:8080 으로 접속하면 빠밤~~ 웹페이지가 뜹니다! 신기신기

프로젝트를 생성하고 나면 src 디렉토리에 이런식으로 구축되어 지더라구요 ㅎㅎ (실질적으로 개발할 src디렉토리)

src
├── assets
│   └── logo.png
├── components
│   └── HelloWorld.vue
├── router
│   └── index.js
├── App.vue
└── main.js

  • assets

    외부에서 가져온 이미지, js, css, font 등을 넣어두는 디렉토리

  • components

    Vue.js에서 사용하는 vue파일들을 생성하고 구현하는 곳이다

  • router/index.js

    서버에서 라우팅을 제공하지 않아도 비슷하게 지원해준다

  • App.vue

    Root컴포넌트로 components에서 만든 것들이 모여서 보여주는 개념..?ㅎㅎ

  • main.js

    프로젝트 베이스 파일로 전역설정 등을 할 수있따~
