1. Vue 컴포넌트 사용하기

1. 전역등록

아래와 같은 형식으로 Vue 인스턴스를 생성했습니다.

new Vue({
	el: '#test'
})

전역 컴포넌트를 사용하려면 아래와 같이 사용합니다

Vue.component('my-component', {
	// 옵션
});

// 이렇게 지정이 되면 <my-component></my-component> 로 사용할 수 있습니다.
// ※꼭 인스턴스를 생성하기 전에 컴포넌트를 먼저 등록 해 주어야 합니다!!※

<div id="com01">
	<my-component></my-component>
</div>

// 등록
Vue.component('my-component', {
	template: '<div>사용자 정의 컴포넌트 입니다!</div>'
})

// 루트 인스턴스 생성
new Vue({
	el: '#com01'
})
여기에 결과가 나오게됩니다~~^^
2. 지역등록

모든 컴포넌트를 전역으로 등록 할 필요는 없겠죠?? components 인스턴스 옵션으로 등록하여 원하는 부분에서만 사용할 수 있어요.

var chiild = {
	template: '<div>사용자 정의 컴포넌트 지역 222</div>'
}

new Vue({
	el: '#com02'
	components: {
		// my-component는 상위 템플릿에서만 사용할 수 있어요
		'my-component': Child
	}
})

3. DOM 템플릿 구문 분석 경고에 대한 처리는?!

DOM을 템플릿으로 사용할때.. 예를들면 table, ul, ol, select 등은 안에 특정한 DOM엘리먼트들만 올 수 있지요?

[table => tr td], [ul, ol => li], [select => option] 등등..

예를들어 테이블 내에 tr, td구분을 template로 사용한다면 어?!이거 테이블형태랑 좀 다른데??1 이상한데?? 하고 삐꾸같이 그려지면서 에러를 냅니다!!

<table>
	<my-row>데이터데이터</my-row>
</table>

// 대신 "is"옵션을 사용합니다

<table>
	<tr is="my-row"></tr>
</table>

4. 데이터는 반드시 함수여야 합니다.

data값은 꼭 함수여야 합니다. 일반적으로 정의하듯이 하면 에러를냅니다

Vue.component('my-com', {
	template: '{{message}}',
	data: {
		message: '아아아'
	}
});
