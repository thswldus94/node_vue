2. Form과 Component

1. 사용자 정의 이벤트를 사용하여 폼 입력 컴포넌트 만들기!?

사용자 정의 이벤트는 v-model 에서 작동하는 사용자 정의 입력값을 만드는데도 사용할 수 있습니당!!

<input v-model="something">

는 아래와 같습니다.

<input v-bind:value="something" v-on:input="something = $event.target.value">

여기서 컴포넌트를 이용한다면?

<custom-input v-bind:value="something" v-on:input="value => { something = value }">
</custom-input>

2. 컴포넌트의 v-model 사용자 정의

컴포넌트의 v-model은 value를 보조 변수로 사용하고 input을 이벤트로 사용해요.
하지만 체크박스와 라디오 버튼 같은 애들은 다른 목적으로 value를 사용할 수 있어요.
model 옵션을 사용하면 충돌을 피할 수 있습니다!

Vue.component('my-checkbox', {
	model: {
		prop: 'checked',
		event: 'change'
	},
	props: {
		// 다른 목적을 위해서 value를 사용할 수 있습니당
		checked: Boolean,
		value: String
	}
})
