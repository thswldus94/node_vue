2. Computed와 Methods

Computed 속성은 종속 대상을 따라 저장(캐싱) 돼요.

이말은 즉, 대상 또는 값이 변경되지 않고 그대로인 경우 함수를 실행하지 않는다는 것을 뜻해요.

여러번 요청 해도, 다시 계산하지 않고 예전에 계산된 결과를 보여준다는 것이죠.

반대로 Methods를 호출하면 호출 할 때마다 계속 함수를 실행해요
