어떠한 속성을 바인드 하기 위해서는 문자열로 계산해야합니다.

근데 매번 문자열로 계산하고 하는 일은 굉장히 빡치는 일이쥬?

이럴때 뷰는 class와 style에 객체와 배열을 이용할 수 있습니다!! 스고이

1. 객체 구문 (HTML)

클래스를 동적으로 toggle하기 위해 v-bind:class 에 객체를 전달할 수 있습니다

이곳은 class toggle 테스트 영역입니다. 테스트로 success 클래스를 넣어봤어요
data: { successClass: true, errorClass: false }

이곳은 class toggle 테스트 영역입니다. 계산을 통해 나온 결과값으로도 설정이 가능합니다 (true, false 등)
data: { classObject: { 'successClass': false, 'errorClass' : true } }
테스트로 danger 클래스를 넣어봤어요
{{ errorMessage }}
2. 배열 구문 (HTML)

배열 속성으로도 v-bind:class에 전달할 수 있어용

바로 이렇게용 :class="[activeClass, errorClass]"

3. 객체 구문 (CSS)

v-bind:style 은 자바스크립트 구문을 인라인 형태로 쓰는거예요

여긴 색깔을 핫핑크로 지정하고 폰트 사이즈를 24px로 지정 해 볼거예요.
4. 배열 구문 (CSS)

v-bind:style 에 여러 스타일의 배열을 사용이 가능해요

여긴 색깔을 흰색으로 지정하고 백그라운드 컬러를 까만색으로, 사이즈는 18px로 해볼게요.
